## 2️⃣ 주요 명령어

1. pwd : 현재 디렉토리 경로
    
2. ls : 파일 정보
    
    a. ls -l : 상세 정보  
    b. ls -a : 숨김 파일
    
3. man : 설명서
    
4. cd : 경로 이동
    
5. mkdir : 폴더 생성
    
6. touch : 파일이 없으면 파일 생성, 있으면 수정과 접근시간을 갱신
    
7. rm
    
    a. rm : 파일삭제  
    b. rm -r : 폴더 삭제
    
8. nano or vim : 파일 수정
    
9. cat : 수정 파일 확인
    
10. mv : 파일을 원하는 폴더로 이동
    
11. cp : 복사
    
12. chmod: 권한 지정
    
13. echo : 한번에 파일 생성하면서 내용도 저장
    
14. grep : 문자열 검색
    
15. sudo : 관리자 권한 필요시 사용
